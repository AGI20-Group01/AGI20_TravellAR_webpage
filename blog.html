Agile 3 - Nov 9

After a long break, we had to refocus our project. Some interesting shaders and light estimation was added. We started to research how to implement haptics and understand when and how to use it. We showcase animation and model of traveler. A few particle effects had been created. Lastly, we showcase the first version of a touch-based movement for the traveler using Unity's navmesh.

Alumni Network Showcase Oct 6

Our first public showcase! We were proud of showing the AR-world working in a server enviroment. During the showcase, we got feedback on the difficulty of using cloud anchors and what type of interaction would work best for the traveler.

Agile 2 - Sep 28

Shows of some progress on textures and adding objects into the world space. Here we can also see the first scene rendered in our AR-world! We tested different input methods for our traveler. We also had to realise that we had to plan our work more to make sure we did not work on the same things.

Agile 1 - Sep 22

We outlined the first ideas for our Game Design. We also show of some basic miscellaneous progress such as server interaction, AR-plane detection, world space management and basic animations.

Proposal - Sep 11

Here, we presented the project as an abstract slide puzzle that we called MystARies. We wished to focus on real-time avatar generation of players and gesture tracking for interaction. None which ended up in the finished product.
